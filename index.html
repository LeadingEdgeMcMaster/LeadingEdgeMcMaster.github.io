Club Overview
Leading Edge solves problems at McMaster and the greater Hamilton area through engaging and innovative projects. Each school year, the club pursues one keystone project and several auxiliary projects with the goal of bridging the gap between theory and application, and between the classroom and the real world, to provide students the opportunity, tools, and mindset necessary to take a project from an idea to a working prototype. There are ample opportunities for students across all skill levels to learn, to build technical skills and to gain experience. All of our projects are student-proposed, which means all general members get a chance to have their ideas considered!

Bluetooth Radar Project
The Bluetooth Radar Project was Leading Edge’s first venture in application-based innovations. Our original goal was to improve accessibilities to libraries and popular study spaces by providing a ‘Bluetooth’ heat map which would account for signal strengths and measurement node’s placements in a facility to get a detailed occupancy map. However the project then evolved to a safer and more cost-effective implementation where the nodes would be placed at entry and exit points to implement a live occupancy measurement of the given study space.
EMG Framework
This year Leading Edge aims to build an Electromyography (EMG) based sensor apparatus which when provided a certain set of gestures will be programmed to execute a certain set of tasks. For example a closed fist to turn off the lights.
PI Vision System
One of Leading Edge’s side projects is to build an image-recognizing vision system. The goal of the system is to provide a base for later projects involving surveillance and security. The vision system would feature facial, image recognition, and infrared sensing for low-light environments.
Game App
Another project running this year is to build a game on both the iOS and Android platforms. This is to provide students an opportunity to learn about App development as well as get their creativity flowing by building an interactive game.
